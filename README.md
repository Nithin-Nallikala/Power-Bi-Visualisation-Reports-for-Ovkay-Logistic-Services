# Power-Bi-Visualisation-Reports-for-Ovkay-Logistic-Services

## Ovkay Logistics Services Analysis Dashboard using Power BI

## Tools Used

MS Excel, Power BI, Power Query, DAX, Data Cleaning, Data Modeling Data Visualization

## Project Description

1. Built an interactive Logistics Analytics dashboard using Power BI to monitor shipments, freight revenue, and shipment weight across multiple Ovkay Logistics branches.
2. Performed extensive data cleaning and transformation using Power Query by integrating multiple Excel datasets, handling null values, trimming data, and creating custom conditional columns.
3. Designed a star-schema data model using fact and dimension tables to improve data organization and enable efficient analysis.
4. Developed calculated columns and DAX measures to derive key business metrics and support dynamic KPI reporting.
5. Created interactive dashboards with KPIs, charts, slicers, and tables to analyze logistics operations across locations, time periods, and shipment routes.
6. Conducted branch-wise, freight-wise, and shipment-weight analysis to identify operational trends and performance patterns.
7. Automated reporting and analysis workflows, contributing to nearly 60% improvement in operational efficiency through faster access to actionable insights.

## Key Insights

1. Identified branch-wise performance differences by analyzing shipment volumes, freight revenue, and shipment weight across logistics centers.
2. Highlighted top-performing branches contributing the highest shipment activity and freight revenue.
3. Tracked shipment trends across months, quarters, and years to identify seasonal and operational patterns.
4. Analyzed source and destination station movements to gain better visibility into logistics flow and transportation networks.
5. Monitored freight revenue and shipment weight together to evaluate operational productivity and business performance.
6. Reduced manual reporting efforts through dashboard automation, improving reporting efficiency and decision-making speed.
7. Enabled stakeholders to monitor critical logistics KPIs from a centralized dashboard and make data-driven operational decisions.
8. Contributed to approximately 60% operational efficiency improvement by streamlining reporting processes and providing actionable business insights.
